Subtotal.js

An open-source JavaScript plugin for PivotTable.js to render rows and columns of a pivot table with subtotals and lets you expand and collapse subtotals.

Pivot Examples

  • With Default Options
  • Collapse Rows and/or Columns On Load
  • Count / Sum As Fraction Of Parent Row - Aggregators
  • Count / Sum As Fraction Of Parent Column - Aggregators
  • Customized Collapse And Expand Characters
  • Mouse Click Callback Handling
  • Hide Row and/or Column Subtotals On Expand
  • Disable Row and/or Column Subtotals
  • Disable Expand Collapse Operations
  • Disable Subtotal Display Afer A Specified Level
